Links for Construct developers

If you're a Construct developer, check out these neat resources made by the community. Tutorials, tips, optimisation techniques, 3rd party tools, et cetera.

Learning resources

  • Construct 3 tips, best practices & notes by James
  • Construct 3 cheat sheets by Federico Calchera
  • Performance tips by Federico Calchera
  • HowToConstruct.com templates & demos
  • Construct 3 tutorials by Skymen
  • Construct 3 video tutorials by Xanderwood
  • GameDesign With Reilly (video tutiorials)
  • Construct 3 tutorials and showcase videos by Foozle

Tools

  • Construct Crawler by Skymen: cross platform app for manipulating Construct projects
  • Pipelab by Armaldio: freemium cross platform app for automating deployment of Construct projects
  • c3p2git by Fodi: Windows CLI utility for turning c3p project files into Git commits
  • Sprite font generators
    • Font To Bitmap by stmn (browser based)
    • SpriteType by Dokse (browser based)
    • Sprite Font Generator by blackhornet (Windows app)
  • Construct Color Converter by Fodi: batch convert hexadecimal RGB palettes into Construct 3 color values & global constant declarations
  • Construct 3 addons by Pixel Perfect Studios
  • Construct Shader Graph by Skymen: a node based visual shader editor with WebGL 1, WebGL 2 and WebGPU support.
  • Google Webfonts Helper: for downloading free font files from Google Fonts in any format (including TTF)
  • CoMiGo's Placeholder Generator: a simple, free and engine-agnostic placeholder sprite generator, great for grayboxing.
  • Javascript PRNGs: public domain code for various seed generators, seeded PRNGs and hashing functions. Can be useful for making games with procedurally generated content.

Services

  • Free(mium) webhosting services for HTML5 builds:
    Itch.io, GameJolt, GitHub Pages, GitLab Pages, Cloudflare Pages, Vercel, Netlify, Surge
  • Game porting services:
    MP2 Games, Ratalaika Games, Seaven Studio
